## Introduction

Markdown is a lightweight, readable syntax for formatting text. In software engineering,
it's the go-to format for documentation, README files, technical blogs, and even
internal communication. This post will demonstrate every major Markdown feature with
examples relevant to our field.

Mastering **Markdown** is an essential skill for developers looking to create clean and
readable documentation. With its simplicity and flexibility, it allows you to craft
everything from README files to blog posts. If you're just starting, check out
[this comprehensive guide](https://www.markdownguide.org) to learn the basics and take
your skills to the next level!

## A color scheme for focusing

**Monokai Pro** is a color scheme, user interface theme and icon pack for code editors.
It has been designed with **beautiful functionality** in mind. Carefully selected shades
of color are the foundation of an uncompromising, non-distractive user interface.

```js showLineNumbers title="filename.js" {3..4,9}
export default class MonokaiPro {
  constructor() {
    this.theme = "Monokai Pro Light"
    this.numIcons = 70
  }

  // set color scheme, UI, icon pack
  setTheme(theme) {
    this.theme = theme

    console.log(`theme: ${theme}`)
  }
}
```

Monokai Pro is available for [Sublime Text](https://www.google.com) and
[Visual Studio Code](https://www.google.com) editors. It comes with predefined syntax
highlighting colors. The single goal of Monokai Pro is to let you **focus on your
code**.
